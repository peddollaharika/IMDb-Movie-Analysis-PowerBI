# 🎬 Movie Industry Analytics Dashboard | Power BI

## 📖 Project Overview

The Movie Industry Analytics Dashboard is an interactive Power BI project developed to analyze movie industry performance using IMDb movie data. The dashboard provides insights into movie releases, box office revenue, audience engagement, genre performance, ratings, metascores, and director performance through visually appealing and interactive reports.

This project demonstrates the use of Business Intelligence techniques to transform raw movie data into meaningful insights that support trend analysis and data-driven decision-making.

---

## 🎯 Objectives

- Analyze movie release trends over time.
- Identify top-performing genres and directors.
- Evaluate IMDb ratings, Metascores, and audience votes.
- Examine box office revenue patterns.
- Discover the most successful movies based on revenue and audience engagement.
- Build an interactive dashboard for efficient data exploration.

---

## 📊 Dashboard Modules

### 1️⃣ Movie Overview & Time Trends

This page provides a high-level overview of movie performance across different years.

**Key Insights**
- Number of movies released each year
- Revenue trend over time
- Average IMDb rating trend
- Average runtime trend
- Top years with highest movie releases
- Average audience votes received each year

---

### 2️⃣ Genre & Rating Analysis

This page focuses on understanding genre popularity and audience preferences.

**Key Insights**

- Revenue generated by each genre
- Average Metascore by genre
- Genres receiving the highest number of votes
- Most common genres in the dataset

---

### 3️⃣ Top Performers & Box Office Insights

This page highlights the most successful movies and directors.

**Key Insights**
- Highest revenue-generating directors
- Most voted movies
- Top grossing movies
- Revenue and rating comparison
- Audience engagement analysis

## 📈 Key Findings

- Movie releases increased significantly over the years.
- 2016 recorded the highest box office revenue.
- Action and Adventure genres generated the highest revenue.
- The Dark Knight received the highest audience votes.
- J.J. Abrams emerged as the highest revenue-generating director.
- Average IMDb ratings remained relatively stable across most years.
- Audience engagement was strongest for highly rated movies.
